# project-2

# Примитивная база данных

Простая база данных для управления таблицами с поддержкой основных операций.

## Управление таблицами

### Команды:

#### Создание таблицы
create_table <имя_таблицы> <столбец1:тип> [столбец2:тип ...]
**Пример:**
create_table users name:str age:int active:bool


#### Просмотр таблиц

list_tables


#### Удаление таблицы
drop_table <имя_таблицы>
**Пример:**
drop_table users


#### Общие команды
help - справочная информация
exit - выход из программы


### Поддерживаемые типы данных:
- `int` - целые числа
- `str` - строки
- `bool` - логические значения

### Особенности:
- Столбец `ID:int` автоматически добавляется в каждую таблицу как уникальный ключ
- Данные сохраняются в файл `database.json`
- Поддерживается корректная обработка ошибок

## Дополнительные возможности

### Обработка ошибок
- Все операции защищены декоратором `handle_db_errors`, который перехватывает:
  - `KeyError` - обращение к несуществующим таблицам или ключам
  - `ValueError` - ошибки валидации типов данных
  - `FileNotFoundError` - отсутствие файлов данных

### Подтверждение действий
- Опасные операции (`drop_table`, `delete`) требуют подтверждения:
Вы уверены, что хотите выполнить "удаление таблицы"? [y/n]:


### Замер времени выполнения
- Операции `insert` и `select` показывают время выполнения:
функция insert выполнилась за 0.125 секунд


### Кэширование запросов
- Результаты одинаковых запросов `select` кэшируются для повышения производительности
