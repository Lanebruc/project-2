# Примитивная база данных

Простая база данных для управления таблицами с поддержкой основных операций.

## Управление таблицами

### Команды:

#### Создание таблицы
create_table <имя_таблицы> <столбец1:тип> [столбец2:тип ...]
**Пример:**
create_table users name:str age:int active:bool


#### Просмотр таблиц

list_tables


#### Удаление таблицы
drop_table <имя_таблицы>
**Пример:**
drop_table users


#### Общие команды
help - справочная информация
exit - выход из программы


### Поддерживаемые типы данных:
- `int` - целые числа
- `str` - строки
- `bool` - логические значения

### Особенности:
- Столбец `ID:int` автоматически добавляется в каждую таблицу как уникальный ключ
- Данные сохраняются в файл `database.json`
- Поддерживается корректная обработка ошибок

### Пример сессии:
DB project is running!
Добро пожаловать в примитивную базу данных!

Введите команду: create_table employees name:str department:str salary:int
Таблица 'employees' успешно создана
